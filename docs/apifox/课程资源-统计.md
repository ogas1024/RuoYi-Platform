# Apifox 指南 - 课程资源统计

- baseURL: `http://localhost:8080`
- 请求头：`Authorization: Bearer <token>`（登录后台获取）

## 步骤 1：获取 token
- 接口：`POST /login`
- Body: `{ "username": "admin", "password": "admin123" }`（按实际账号）
- 断言：`code == 200`，保存 `token` 备用

## 步骤 2：上传趋势（按日）
- 接口：`GET /manage/courseResource/stats/uploadTrend`
- Headers：`Authorization: Bearer {{token}}`
- Params：`days=30`
- 断言：
  - 状态码：200
  - 响应：`code == 200`
  - 响应体：`data` 为数组，元素包含 `day`、`count`

## 常见失败与处理
- 403 无权限：在 RuoYi 后台为角色分配权限标识 `manage:courseResource:stats`；重新登录后重试。
- data 为空：近 N 天没有上传记录（预期情况）。
