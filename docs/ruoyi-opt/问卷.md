# RuoYi 后台操作指引：问卷管理菜单

1. 登录 RuoYi 后台（管理员）。
2. 系统管理 → 菜单管理 → 新增
   - 菜单类型：目录
   - 菜单名称：表单收集
   - 路由地址：`/manage/survey`
   - 权限标识：无（目录无需）
3. 在“表单收集”目录下继续新增
   - 菜单类型：菜单
   - 菜单名称：问卷管理
   - 路由地址：`/manage/survey/index`
   - 组件路径：`views/manage/survey/List`
   - 权限标识：`manage:survey:list`
   - （可选）默认查询参数：无（界面默认展示“已发布”）
4. 在“表单收集”目录下新增“归档列表”菜单
   - 菜单类型：菜单
   - 菜单名称：归档列表
   - 路由地址：`/manage/survey/archive`
   - 组件路径：`views/manage/survey/List`
   - 权限标识：`manage:survey:list`
   - 备注：该路由自动切换为“已归档”筛选
5. 在“表单收集”目录下新增“新建表单”菜单
   - 菜单类型：菜单
   - 菜单名称：新建表单
   - 路由地址：`/manage/survey/new`
   - 组件路径：`views/manage/survey/Create`
   - 权限标识：`manage:survey:add`
6. 新增按钮权限（同属“问卷管理”菜单）
   - 按钮：查看详情 → 权限标识：`manage:survey:query`
   - 按钮：新建 → 权限标识：`manage:survey:add`
   - 按钮：归档 → 权限标识：`manage:survey:archive`
   - 按钮：延期 → 权限标识：`manage:survey:extend`
   - 按钮：置顶/取消置顶 → 权限标识：`manage:survey:pin`
   - 按钮：编辑（仅草稿） → 权限标识：`manage:survey:edit`
   - 按钮：发布（仅草稿） → 权限标识：`manage:survey:publish`
7. 角色授权
   - 系统管理 → 角色管理 → 选择管理员/运营角色 → 勾选“表单收集/问卷管理”及其按钮权限。
   - 用户管理 → 将测试用户分配至上述角色，重新登录生效。

前端门户无需后台菜单：访问路径在前端路由为 `/portal/survey`、`/portal/survey/fill`、`/portal/survey/my`。
