# RuoYi 后台操作教程：首页报表（课程资源上传趋势）

目的：确保首页 `/index` 折线图能成功调用统计接口。当前策略：只要“已登录后台”，即可访问统计接口（无需额外按钮权限）。

## 一、权限规则（当前）
- 统计接口放宽为“登录可见”：
  - 课程资源：`/manage/courseResource/stats/uploadTrend|downloadTrend`
  - 数字图书：`/manage/library/stats/uploadTrend|downloadTrend`
- 无需新增按钮权限；拥有后台登录账号即可在首页看到报表。

## 二、验证
- 打开首页：`/index`
- 期望：课程资源与数字图书两组折线图均能显示；无数据时坐标轴为空。
- 若出现 401：检查登录状态（token）。

## 四、接口速查
- 路径：`GET /manage/courseResource/stats/uploadTrend`
- 权限：登录（isAuthenticated）
- 参数：`days`（可选，默认30，范围1~365）
- 返回：`[{ day: 'yyyy-MM-dd', count: number }]`（已按天补零、升序）

更多细节参见：`docs/api/课程资源-管理.md` 与 `docs/api/数字图书馆-管理.md` 里的“统计趋势”章节。

---
扩展位：后续在同一页面可继续添加“通知公告、功能房预约、数字图书馆”等模块的统计图表。若需更细权限，可改回按钮级控制（例如 `manage:<module>:stats`）。
