# RuoYi 后台操作指引：投票管理菜单（独立于问卷）

1. 登录 RuoYi 后台（管理员）。
2. 系统管理 → 菜单管理 → 新增目录
   - 菜单类型：目录
   - 菜单名称：投票管理
   - 路由地址：`/manage/vote`
   - 权限标识：无
3. 在“投票管理”目录下新增菜单：投票列表
   - 菜单类型：菜单
   - 菜单名称：投票列表
   - 路由地址：`/manage/vote/index`
   - 组件路径：`views/manage/vote/List`
   - 权限标识：`manage:survey:list`
4. 在“投票管理”目录下新增菜单：投票归档
   - 菜单类型：菜单
   - 菜单名称：投票归档
   - 路由地址：`/manage/vote/archive`
   - 组件路径：`views/manage/vote/List`
   - 权限标识：`manage:survey:list`
   - 说明：该路由自动切换“已归档”筛选
5. 在“投票管理”目录下新增菜单：新建投票
   - 菜单类型：菜单
   - 菜单名称：新建投票
   - 路由地址：`/manage/vote/new`
   - 组件路径：`views/manage/vote/Create`
   - 权限标识：`manage:survey:add`
6. 按钮权限（归属“投票列表”菜单）
   - 查看详情：`manage:survey:query`
   - 新建/编辑：`manage:survey:add`, `manage:survey:edit`
   - 发布：`manage:survey:publish`
   - 延期：`manage:survey:extend`
   - 归档：`manage:survey:archive`
   - 置顶/取消置顶：`manage:survey:pin`
7. 角色授权
   - 系统管理 → 角色管理 → 选择运营/管理员 → 分配上述菜单与按钮权限。
   - 用户管理 → 给测试账号分配角色，重新登录生效。

说明
- 投票后端复用问卷接口，因此权限标识仍为 `manage:survey:*`。
- 门户“投票”入口位于 `/portal/vote`，不依赖后台菜单。
