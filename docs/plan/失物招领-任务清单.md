# 失物招领｜任务清单（MVP）

- 基线：一律先审核；编辑回到待审；已解决不可回退；回收站可恢复为待审；图片≤9。

## 后端（ruoyi-manage）
- [ ] Domain：LostItem、LostItemImage（含校验注解）
- [ ] Mapper：LostItemMapper.xml、LostItemImageMapper.xml
- [ ] Service：ILostItemService + LostItemServiceImpl（含状态校验与图片保存事务）
- [ ] Controller（门户）：/portal/lostfound（list/get/add/update/delete/solve/my）
- [ ] Controller（管理）：/manage/lostfound（list/audit/recycle/get/approve/reject/offline/restore/add/edit/remove）
- [ ] 权限注解：`manage:lostfound:*`（与文档一致）
- [ ] Bean 校验与统一错误码（409 冲突、400 参数）
- [ ] 单元/集成测试（成功/失败/权限）

## 前端（Vite + Vue3 + Element-Plus）
- [ ] API：portal/lostfound.js；manage/lostfound.js
- [ ] 门户：views/portal/lostfound/List.vue、Edit.vue、My.vue
- [ ] 管理：views/manage/lostfound/List.vue、Audit.vue、Recycle.vue
- [ ] 图片上传：复用 `POST /manage/upload/oss`；表单组件限制 ≤9 张
- [ ] 按钮权限：v-hasPermi 与后端一致
