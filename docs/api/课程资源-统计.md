# 课程资源分享模块 - 统计 API

- 概览：提供课程资源上传数量的趋势统计（按日）。用于管理后台首页折线图展示。
- 角色权限：需要权限标识 `manage:courseResource:stats`（管理员默认具备）。
- 依赖表：`tb_course_resource`

## 路由清单

### 接口：上传趋势（按日）
- 方法：GET
- 路径：`/manage/courseResource/stats/uploadTrend`
- 权限：`manage:courseResource:stats`
- 请求参数（query）：
  - `days` (int) 可选，默认 30；统计最近 N 天（含今天）。范围 1~365，超范围按 30 处理。
- 返回示例：
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": [
    { "day": "2025-10-10", "count": 3 },
    { "day": "2025-10-11", "count": 0 },
    { "day": "2025-10-12", "count": 5 }
  ]
}
```

备注：
- 统计口径为 `tb_course_resource.create_time`（上传即记录），软删数据不参与统计（`del_flag='0'`）。
- 返回数据已按天补零，且按日期升序排列。
